## How to use

Easy use. Download MMRPGDigitalCharacterSheet.html. To allow easy homebrew and to avoid copyright, the sheet is initially empty
For functional use, import a library(ies) (a .json file that contains origins, traits, tags, occupations, powers, items, etc.)

Find through one of 3 ways:
  1. Find someone else who already created one
  2. Use the sampleLibrary.json which only contains elements from the Free Basic Rules
  3. Use the MMRPGDigitalLibraryEditor.html to create your own

Store your character through one of 3 ways:
  1. Save as .mar (a small file that requires the library used to create it to be functional)
  2. Save as .marvel (a larger file that is self-contained with its library logic and rules)
  3. Export as .pdf (using MARRPGP_Character_Sheet_Color_Fillable.pdf as a template)
